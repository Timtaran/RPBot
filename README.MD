# Запуск бота
Для запуска бота необходимо его запускать следующим образом:
**Установка библиотек:**

Windows:
```cmd
 pip install -r requirements.txt
```
Linux/MacOS:
```console
python3 -m pip install -r requirements.txt
```
**Запуск:**

Windows:
```cmd
py main.py bot_token
```
Linux/MacOS:
```console
python3 main.py bot_token
```
